# 01. HOW THIS COURSE WILL WORK

## Introduction

The Odin Project is an open-source community, dedicated to providing the best information sources to take you from zero to a full-stack developer.

In this unit we'll learn:

- How the web works
- About the basics of computer and web programming

We will start with:

- Getting familiar with the internet and your own computer
- Setting up a development environment
- Learning about Git and Github
- Going over the basics of HTML, CSS, and Javascript
- A brief foray into the basic concepts of back-end

By the end of this unit, you should:

- Understand how the web works
- Be able to identify and differentiate between web technologies
- Be able to:
    - Build a basic webpage
    - Style it
    - Add elements of interactivity
    - Work from the command line

## How it works

The curriculum works by aggregating content on specific topics. Each lesson introduces the context, then points you to external resources.

Lessons may include:

- Questions you should be able to answer before moving on
- Exercises for you to complete

Additionally, there are several projects where you actually build things. The projects
are not tests of what you have memorised so far; they are practice points.

## A note about tools

- The TOP community is often asked "Why not X tool?"
- The point of the curriculum is to get you to a level where shouldn't need something like
Odin to learn X tool
- The main idea is to teach you the WHY and change the way you think
