# 07. TEXT EDITORS

## Introduction

- Text editors are by far the most used developer tool
- They can help you write better code with:
    - Real-time code checking
    - Syntax highlighting
    - Automatic formatting

## Code editors

- Can be thought of as specialised development tools
- Highly customisable, with features that make your life easier
- Benefits include plugins, syntax highlighting, lintin, auto-closing brackets and braces

There are many editors out there, bu Odin suggest VS Code. Note that this is the sole
editor that TOP supports.
