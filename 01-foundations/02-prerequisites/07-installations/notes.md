# 07. INSTALLATIONS

## Introduction
The first step in building any website is having the right tools. For us, that means
setting up a development environment for writing good code.

Many online development courses use in-browser code editors or "sandboxes", which give you
the tools and programs needed to accomplish the task at hand and nothing else. You'll use
some of these sandboxes throughout the early stages of The Odin Project since they're great
for getting started quickly. However, the best way to set yourself up for long-term
success is to operate in a real development environment.

Installing packages, editors, and even entire operating systems can be frustrating.

However, having the experience of setting up a development environment to run the code
you'll write is an invaluable, real-world skill you'll carry with you for the rest of your career.
